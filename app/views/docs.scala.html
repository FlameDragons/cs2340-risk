Deployment

CS 2340 Risk is deployed within a Docker container on a bare metal server running Ubuntu Server. The container itself is based on the openjdk:8-jre-alpine image, which runs a Java 8 JRE on an Alpine Linux OS. The entire application is packaged in a single docker image hosted on DockerHub, which can be deployed by running the following commands: 

$ sudo docker pull jazevedo6/cs2340-risk:latest
$ sudo docker run -it -p 9000:9000 -p 9443:9443 --name <name> jazevedo6/cs2340-risk:latest
By default, the application exposes ports 9000 and 9443 for HTTP and HTTPS respectively.
